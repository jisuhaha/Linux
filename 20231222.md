# FPT Server

### FPT Service 동작원리
1. Active 모드
> 클라이언트에서 서버 포트를 통해 접속 시도 이후 응답에서 사용할포트 제공받음
	서버에서 클라이언트로 해당포트로 연결하여 데이터 전송
	이로인해 방화벽으로인해 파일전송이 제한될 수 있음
2. Passive 모드
> 클라이언트에서 서버 포트를 통해 접속 시도 이후 응답에서 사용할포트 제공받음
	클라이언트에서 서버로부타 응답받은 포트로 연결
