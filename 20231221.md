# Apache WebServer
	버전정보 가리기
	서비스의 파일httpd.conf에서 Server Tokens값 변경 
	>ServerTokens Prod
	ServerTokens : Server 클라이언트로 다시 전송되는 응답 헤더 필드에 	서버의 일반 OS 유형에 대한 설명과 컴파일된 모듈에 대한 정보가 포함되는지 여부를 제어
	
	example
	full : Apache/2.4.2 (Unix) PHP/4.2.2 MyMod/1.2
	prod : Apache/2
	Major : Apache/2 
	Minor : Apache/2.4
	Min : Apache/2.4.2
	OS : Apache/2.4.2 (Unix)
